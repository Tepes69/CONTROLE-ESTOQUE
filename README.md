# Controle de Estoque
Projeto desenvolvido em Python para registrar uma movimentação de entrada ou saída
e apresentar a quantidade atualizada de um produto.
## Funcionalidades
- Solicitar o nome do produto;
- Registrar a quantidade inicial;
- Registrar entrada ou saída;
- Calcular o estoque atualizado;
- Impedir uma saída maior que o estoque disponível.
## Tecnologias utilizadas
- Python
- Git
- GitHub
